# Flexo Print Website

## Описание проекта

Flexo Print Website — это веб-приложение, разработанное для компании,
занимающейся флексографической печатью.

Цель проекта — создать современный веб-сайт, который позволит компании
представлять свои услуги, демонстрировать портфолио работ и принимать
заявки от клиентов через интернет.

Веб-сайт предназначен для повышения удобства взаимодействия с клиентами,
расширения клиентской базы и улучшения онлайн-присутствия компании.

## Архитектура проекта

В проекте используется **многослойная архитектура (Layered Architecture)**.

Система разделена на несколько уровней:

### Frontend

Отвечает за пользовательский интерфейс и взаимодействие с пользователем.

### Backend

Обрабатывает запросы пользователей и реализует бизнес-логику приложения.

### Database

Отвечает за хранение данных приложения.

Взаимодействие компонентов:
Пользователь → Frontend → Backend → Database

## Установка зависимостей

Перед запуском проекта необходимо установить зависимости.

### Backend

cd cd backend
node server.js
npm install

### Frontend

cd frontend
npm install

---

## Запуск проекта

### Запуск backend

cd backend
node server.js

### Запуск frontend

cd frontend
npm start
После запуска сайт будет доступен по адресу:
http://localhost:3000

## Сборка проекта

Для сборки frontend-части:
